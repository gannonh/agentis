# Product Overview

Agentis is an all-in-one AI conversations platform that integrates multiple AI models into a single chat interface. It's a branded fork of LibreChat with custom capabilities.

## Core Features
- Multi-model AI chat interface supporting Anthropic Claude, OpenAI, Azure OpenAI, Google Vertex AI, and custom endpoints
- Real-time streaming responses with WebSocket/SSE support
- Code execution environment via integrated Sandpack
- Document processing and retrieval (RAG) capabilities
- User authentication and session management
- Role-based access control and user management
- Model Context Protocol (MCP) integration for enhanced AI capabilities

## Key Integrations
- **Self-hosted dependencies**: RAG API and Sandpack code execution environment
- **Authentication**: JWT-based stateless authentication with Better Auth
- **Database**: MongoDB for data persistence
- **Real-time**: WebSocket/SSE for streaming AI responses
- **File handling**: Support for document upload and processing

## Target Users
Developers and organizations seeking a unified interface for multiple AI models with advanced features like code execution and document processing.
